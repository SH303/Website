(15/04/22) Design Decisions:

Whilst developing my website over the term I had to really consider the design. For the background I wanted to be unique I didn’t want to just use colours that’s why I chose to use a GIF for the background as it is moving and repeated. I also used Tahoma font instead of the default font as I didn’t want the website to look too generic. For the colours I didn’t go too overboard and used basic colours, but I made my contact form a dark theme as it flows with the GIF in the background, I mostly used normal colour names but for the form I used hex as it was from a lecture slide and activity.

(15/04/22) Changes:

First, I had it in body then changed it so I could have different grid layouts for different pages, so I made div classes for this in each HTML file and designed it in CSS.
